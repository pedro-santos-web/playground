## About Playground

Playground is a simple app to get info from domains, ip addresses and save hosts that you would like to keep an eye on.
It has the following features:

- Add hosts
- Edit hosts
- Delete hosts
- Check URL without saving
- Check IP
- Check IP location

## Tech used

- PHP
- Laravel
- Blade
- Tailwind CSS
- Javascript
- jQuery
- Leaflet
- OpenStreetMap
- IPinfo API
